# [Linux 101 Docs](https://101.lug.ustc.edu.cn/)

## 如何构建？

1. （建议）使用 `python3 -m venv venv` 创建虚拟环境，`. venv/bin/activate` 进入虚拟环境；
2. 使用 `pip install -r requirements.txt` 安装依赖；
3. 使用 `mkdocs serve` 构建并查看效果。
